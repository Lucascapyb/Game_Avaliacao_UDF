# Game_Avaliacao_UDF
**Repository dedicated to the development of a ludic prototype for disciplinary evaluation.**

### Project Overview
This project consists of a technical study developed during the early stages of my academic background in Computer Science. The main objective was to consolidate fundamental concepts of programming logic and system architecture focused on digital entertainment, serving as an evaluative requirement for one of the undergraduate courses.
---
### System Components and Features (HUD & Gameplay)

   ### The application integrates the following modules and resources:<br>
<div>
  <ul>
    <li>Implementation of collision logic and response to stimuli between entities;<br></li>
    <li><code>fight</code> Combat Systems: Implementation of collision logic and response to stimuli between entities;<br></li>
    <li><code>status</code>: State Management (Health System): Interface and logic for monitoring character integrity;<br></li>
    <li><code>boss</code>: Final Challenge Instance (Boss Fight): Implementation of AI behavior patterns and victory conditions.<br></li>
    <li><code>animation</code>: Animation Module: Manipulation of sprites and visual transition states;<br></li>
    <li><code>environment</code>: Environment Composition: Scenery structuring and basic level design;<br></li>
  </ul>
</div>
